# Contributing

Terima kasih sudah ingin membantu.

## Aturan Nama & Kredit

- Mohon sertakan kredit ke proyek ini saat redistribusi atau membuat turunan.
- Gunakan teks kredit berikut:
  - "Powered by bheibzcolection"
  - "Dikerjakan oleh bheibz digital meedia dan Codex"

## Update Referensi

Jika kamu menambahkan atau mengganti referensi repo di `repos.md`, mohon:
- Sertakan alasan singkat di deskripsi PR.
- Pastikan format sesuai (URL GitHub, `cmd:`, atau `deps:`).

## Cara Kontribusi

1. Fork repo ini.
2. Buat branch baru.
3. Lakukan perubahan.
4. Buat Pull Request.

Terima kasih.
